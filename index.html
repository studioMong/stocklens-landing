서비스 소개

StockLens(이하 "서비스")는 미국 주식 시장에 관심 있는 개인 투자자를 위해 SEC 공시 데이터, 재무제표, 시장 감성 지수, AI 교차 분석 기능을 제공하는 정보 플랫폼입니다.

운영사 정보

  • 서비스명: StockLens
  • 운영 주체: moodilab
  • 문의: support@stocklens.app
  • 서비스 URL: https://stocklens-production-3724.up.railway.app

제공 기능

  • 미국 상장 종목 기본 프로필 및 주가 정보 (Financial Modeling Prep API 기반)
  • 최근 재무제표 4개 분기 데이터 (매출, 영업이익, 순이익)
  • 뉴스 헤드라인 및 시장 감성 분석 (Finnhub API 기반)
  • SEC EDGAR Form 4 내부자 거래 공시 데이터
  • 기관투자자 13F 보고서 분석 (Oracle 플랜)
  • Claude AI 기반 종목 교차 분석 (Analyst, Oracle 플랜)

투자 고지사항

StockLens는 「자본시장과 금융투자업에 관한 법률」상 투자자문업자 또는 투자일임업자가 아닙니다. 본 서비스에서 제공되는 모든 정보는 투자 권유 또는 추천이 아닌 순수 정보 제공 목적입니다.

데이터 정확성

본 서비스는 외부 API(Financial Modeling Prep, Finnhub, SEC EDGAR)를 통해 데이터를 수집하며, 데이터의 정확성, 완전성, 적시성을 보증하지 않습니다. 투자 결정 전 반드시 원본 공시 자료를 확인하시기 바랍니다.

AI 분석 한계

AI(Claude)가 제공하는 분석은 과거 데이터 기반의 패턴 인식이며, 미래 주가를 예측하지 않습니다. AI 분석 결과에는 오류가 포함될 수 있으며, 이를 유일한 투자 근거로 활용해서는 안 됩니다.

금지 표현

본 서비스의 AI 분석 기능은 「자본시장법」을 준수하여 매수(buy), 매도(sell), 추천(recommend), 투자 권유(invest) 등의 표현을 자동 필터링합니다. 서비스 내 모든 표현은 객관적 데이터 서술에 한정됩니다.
